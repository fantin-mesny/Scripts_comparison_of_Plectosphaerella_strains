# Comparative analysis of 72 *Plectosphaerella* strains

This repository includes scripts and data used as part of analyses in the following publication:
Raja-Kumar, R. S., Mesny, F., Basak, A. K., Newfeld, J., Chesneau, G., Entila, F., ... & Hacquard, S. (2025). Genetic factors driving multi-host infection in a core member of the root mycobiota. *bioRxiv*, 2025-12. doi: [10.1101/2025.12.01.690973](https://doi.org/10.1101/2025.12.01.690973)

 - Analysis of the ecology of core mycobiota members: [GlobalFungi analysis](https://github.com/fantin-mesny/Scripts_comparison_of_Plectosphaerella_strains/tree/main/GlobalFungi)
 - Phylogenetic tree reconstruction from *Plectosphaerella* genomes: [Phylogeny reconstruction](https://github.com/fantin-mesny/Genomic-signatures-of-host-adaptation-in-a-ubiquitous-fungal-endophyte/tree/main/Phylogenetic%20reconstruction)
 - Analysis of genome transposon annotations: [Transposon analysis](https://github.com/fantin-mesny/Scripts_comparison_of_Plectosphaerella_strains/tree/main/Transposons)
 - Analysis of gene conservation across the genomic dataset and of gene proximity to transposons: [Gene Conservation Analysis](https://github.com/fantin-mesny/Scripts_comparison_of_Plectosphaerella_strains/tree/main/GeneConservationAnalysis)
 - Analysis (PCoA+PERMANOVA) of CAZyme, protease, effector and orthogroup repertoires: [Gene Repertoire Analysis](https://github.com/fantin-mesny/Scripts_comparison_of_Plectosphaerella_strains/tree/main/GeneRepertoireCompositions)
 - Analysis of plant phenotypes (shoot fresh weights) after plant inoculation with the 72 *Plectosphaerella* isolates: [Plant Phenotype Analysis](https://github.com/fantin-mesny/Scripts_comparison_of_Plectosphaerella_strains/tree/main/PlantInoculationExperiments)
 - Analysis of *Plectosphaerella* growth at high temperature: [Temperature](https://github.com/fantin-mesny/Scripts_comparison_of_Plectosphaerella_strains/tree/main/Temperature)
